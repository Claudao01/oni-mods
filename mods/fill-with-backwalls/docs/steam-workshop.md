# Texto para a Steam Workshop

## Português (Brasil)

### Fill with Backwalls

Preencha uma cavidade fechada com paredes de fundo usando um único clique. O mod utiliza o modelo, o material e o visual atualmente selecionados e também pode substituir paredes existentes.

**Como usar**

1. Selecione uma parede de fundo no menu de construção.
2. Escolha o modelo, o material e o visual.
3. No painel de modelos e materiais da parede de fundo, marque o checkbox exibido ao lado ou abaixo da lista de materiais. Em pt-BR ele aparece como **Preencher cavidade**; o nome pode variar conforme o idioma ativo.
4. Clique dentro de uma cavidade fechada.

O tamanho máximo da cavidade pode ser alterado em **Mods > Fill with Backwalls > Configurar**. O valor padrão é 120 células. A tela de configuração possui um link para o repositório do mod: https://github.com/Claudao01/oni-mods/tree/main/mods/fill-with-backwalls

O mod inclui traduções para pt-BR, alemão, espanhol, francês, russo, chinês simplificado, japonês e coreano, com inglês como fallback.

Compatível com a API 2 de mods do ONI. A compatibilidade final com jogo base e Spaced Out! deve ser confirmada antes da publicação.

## English (US)

### Fill with Backwalls

Fill an enclosed cavity with backwalls in a single click. The mod uses the currently selected blueprint, material, and facade, and can also replace existing backwalls.

**How to use**

1. Select a backwall from the build menu.
2. Choose the blueprint, material, and facade.
3. In the backwall blueprint and material panel, enable the checkbox displayed next to or below the material list. Its label follows the active game language; in English it is **Fill cavity**.
4. Click inside an enclosed cavity.

The maximum cavity size can be changed under **Mods > Fill with Backwalls > Configure**. The default is 120 cells. The configuration screen includes a repository link: https://github.com/Claudao01/oni-mods/tree/main/mods/fill-with-backwalls

The mod includes pt-BR, German, Spanish, French, Russian, Simplified Chinese, Japanese, and Korean dictionaries, with English as the fallback.

Compatible with ONI Mod API 2. Final compatibility with the base game and Spaced Out! must be confirmed before publishing.
