# Changelog

## 1.0.0 - 2026-07-13

- Primeira versao publica estavel.
- Randomizacao independente de nome, cabelo, olhos, boca e conjunto coerente de pele.
- Pools isolados para duplicantes Standard e Bionic.
- Roupas, outfits e trajes permanentemente fora do escopo.
- Configuracoes globais usadas como defaults para novas partidas.
- Provider de overrides por save compativel com o host publico `per-save-mod-settings`.
- Preview animado, exportacao CSV, diagnostico opt-in e log de erros timestampado.
- Botao opcional para rerrolar somente a aparencia fisica na Printing Pod.
