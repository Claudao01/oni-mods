# Tutorial de modding para Oxygen Not Included

Este diretório reunirá materiais práticos para criar, testar, depurar e publicar mods para **Oxygen Not Included**.

Os tutoriais serão organizados por etapa, começando pela preparação do ambiente e avançando para referências do jogo, patches Harmony, interface, configurações, localização, empacotamento e publicação.

O conteúdo será incremental e focado em exemplos reproduzíveis. Cada guia deverá informar pré-requisitos, arquivos envolvidos, como validar o resultado e limitações conhecidas.

Para usar um mod já publicado, consulte o README específico dentro de `mods/`.
